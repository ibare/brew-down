# Brew Down

**macOS용 마크다운 뷰어**

## 기능

- 마크다운 파일 열기
- GFM (GitHub Flavored Markdown) 지원
- 다크 모드 자동 지원
- 드래그 앤 드롭

## 코드 예시

```javascript
const greeting = "Hello, World!";
console.log(greeting);
```
